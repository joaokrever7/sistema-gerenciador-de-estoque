
# Sistema Gerenciador de Estoque

## Descrição

O **Sistema Gerenciador de Estoque** é uma aplicação desenvolvida para facilitar o controle e a gestão de produtos em estoque, ideal para pequenas e médias empresas. Com uma interface simples e funcionalidades completas, permite cadastrar produtos e controlar entradas e saídas.

---

## Funcionalidades

- Cadastro, edição e remoção de produtos
- Controle de entrada e saída de estoque
- Visualização do saldo atual de cada produto
- Interface intuitiva e fácil de usar

---

## Tecnologias Utilizadas

- Linguagem: Python
